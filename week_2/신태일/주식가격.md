## 🧩 문제 설명

- [주식가격](https://school.programmers.co.kr/learn/courses/30/lessons/42584)

### 💡 풀이 과정 (Algorithm)

1. **첫번째 인덱스 / 나머지 인덱스 이 두개를 비교**

- 헬퍼 함수의 인자로 아싸리 비교군 index도 보내는게 나을듯

2. **헬퍼함수 내에서 비교군 index 이후 index만을 반복문 돌리기**

- 카운트를 선위증가시키다가 비교값보다 작으면 break하자

3. **prices를 반복문 돌리면서 정답 return 시켜버리기**

### 💻 코드 (Code)

```javascript
function solution(prices) {
  let 정답 = [];

  for (let i = 0; i < prices.length; i += 1) {
    정답.push(compareTarget(prices, i));
  }

  return 정답;
}

function compareTarget(targetArr, startIndex) {
  let cnt = 0;
  let basePrice = targetArr[startIndex];

  for (let i = startIndex + 1; i < targetArr.length; i += 1) {
    cnt += 1;
    if (basePrice > targetArr[i]) {
      break;
    }
  }

  return cnt;
}
```

### 🏁 회고 (Retrospective)

- **효율성**: 이중 반복으로 풀었기에 O(N^).. 별로 좋지 않은 효율성이다. 이것도 담에 다시 한 번 도전해보자

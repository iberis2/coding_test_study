## 🧩 문제 설명

- [짝지어 제거하기](https://school.programmers.co.kr/learn/courses/30/lessons/12973)

### 💡 풀이 과정 (Algorithm)

1. "문제 제목부터 짝을 지어서 제거한다?"

- 문제가 나에게 스택 활용하라고 콘서트하는 수준

2. "스택 push pop"

- 어차피 중간꺼가 지워지면 그 앞 뒤도 연쇄적으로 지워질텨

3. "늘 그렇듯이 스택 length로 판별"

### 💻 코드 (Code)

```javascript
function solution(s) {
  let stack = [];

  for (let i = 0; i < s.length; i += 1) {
    if (stack.length > 0 && stack[stack.length - 1] == s[i]) {
      stack.pop();
    } else {
      stack.push(s[i]);
    }
  }

  return stack.length > 0 ? 0 : 1;
}
```

### 🏁 회고 (Retrospective)

- **효율성**: 딱 한 번만 돌면 되니까 O(N)
